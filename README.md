# Fore! Mini Golf Support

  Having trouble or have feedback? We'd love to hear from you.

  **Email**: ortolf.c@gmail.com

  **FAQ**

  **Q: The app is crashing. What do I do?**
  A: Make sure you're running the latest version of iOS and the app. Try deleting and reinstalling.

  **Q: How do I unlock new levels?**
  A: Complete all 5 holes in a level to unlock the next one.

  **Q: Is the game free?**
  A: Yes, completely free with no ads or in-app purchases.
